# 🧠 Data-Driven Healthcare Analytics

## 🔥 Pourquoi ce projet ?

Dans le secteur de la santé, les données sont une mine d’or… encore faut-il savoir les exploiter.  
Ce projet plonge au cœur d’un jeu de données hospitalier de **10 000 patients** pour en extraire des tendances claires, des corrélations surprenantes et des insights actionnables.

> 🎯 Objectif : **Transformer des données brutes en intelligence décisionnelle** pour la gestion des soins, des assurances et des ressources médicales.

---

## 📊 Ce que j’ai analysé

| Domaine | Variables clés |
|--------|----------------|
| 👤 Démographie | Âge, sexe, groupe démographique |
| 🩺 Clinique | Pathologie, médicament, résultats de tests |
| 🏥 Logistique | Type d’admission, durée d’hospitalisation |
| 💰 Financier | Montant facturé, assureur |
| 🧬 Biologique | Groupe sanguin |

---

## 🧹 Travail réalisé

- ✅ Nettoyage avancé des données (valeurs manquantes, incohérences, doublons)
- ✅ Feature engineering pour créer des groupes d’âge, groupes sanguins normalisés, etc.
- ✅ Analyse croisée (tables de contingence, heatmaps, graphiques empilés)
- ✅ Visualisations claires et interprétables (Matplotlib, Seaborn)

---

## 🔍 Résultats marquants (exemples)

- 📌 **L’hypertension** est la pathologie la plus fréquente, touchant majoritairement les hommes d’âge moyen.
- 📌 **Medicare** est l’assureur dominant chez les seniors, tandis que **UnitedHealthcare** est plus présent chez les actifs.
- 📌 Les patients souffrant d’**obésité** restent plus longtemps hospitalisés (jusqu’à 14 jours en moyenne).
- 📌 **Corrélation notable** entre groupe sanguin O‑ et certaines pathologies chroniques.

> 💡 Ces résultats peuvent aider à orienter les campagnes de prévention, optimiser les contrats d’assurance ou ajuster les plans de capacité hospitalière.
